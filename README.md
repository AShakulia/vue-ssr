# Инструкция по развертыванию проекта

## Предварительные требования

Перед началом убедитесь, что у вас установлены следующие инструменты:

- [Node.js](https://nodejs.org/) (версия 16 или выше)
- [npm](https://www.npmjs.com/) (обычно устанавливается вместе с Node.js)

## Шаг 1: Клонирование репозитория

Склонируйте репозиторий с проектом на ваш локальный компьютер:

```bash
git clone https://github.com/AShakulia/vue-ssr.git
cd vue-ssr
```

## Шаг 2: Установка зависимостей

Установите все необходимые зависимости, выполнив команду:

```bash
npm install
```

## Шаг 3: Запуск проекта в режиме разработки
Для запуска проекта в режиме разработки выполните команду:

```bash
npm run dev
```

## Шаг 4: Сборка проекта
Для сборки проекта выполните команду

```bash
npm run build
```

### Шаг 5: Запуск проекта в production режиме
Для запуска проекта в production режиме выполните команду:

```bash
npm run preview
```
